FAQ

Frequently Asked Questions

Is this tool really free?
Yes, completely free. No signup, no watermark, no hidden fees. The AI background removal uses remove.bg's API which is included.
Which background color should I use for passport photos?
Most countries (Pakistan, UK, US, Schengen) require a plain white or off-white background. Some countries allow light blue. Always check the specific requirements for your application.
Are my photos uploaded to a server?
Only when using the "Remove Background" AI feature — your image is sent to remove.bg's API for processing. This is required for AI background removal. If you use "Keep Original", everything runs locally in your browser.
What size should I choose for Pakistan passport?
For Pakistani passport and CNIC, use the "Pakistan Standard" preset which is 35×45 mm (413×531 pixels at 300 DPI). Use white background color.
Can I use this for printed passport photos?
Yes. Download in JPG or PNG format and take to any photo print shop. The standard presets output at print-ready resolution suitable for official documents.
