FAQ

Frequently Asked Questions

Does resizing reduce image quality?
Downscaling (making smaller) typically preserves visual quality very well. Upscaling (making larger) can result in some blur or pixelation since you're adding pixels that didn't originally exist. For best results, avoid upscaling more than 2x.
Does aspect ratio lock work for all images?
Yes — when Lock Ratio is enabled and you enter only one dimension, the other is calculated automatically based on the original image's aspect ratio. If you enter both dimensions with Lock Ratio on, it will use both values as-is.
Are my images uploaded to a server?
Never. All resizing happens directly inside your browser using the HTML5 Canvas API. Your images never leave your device, so your files are completely private and secure.
Can I resize to exact pixels?
Yes — enter exact pixel values in the Width and Height fields. Disable "Lock ratio" if you want to set both dimensions independently without automatic calculation.
Which output formats are supported?
You can keep the same format as the original, or convert to JPG, PNG, or WEBP during resize. This lets you resize and convert in one step.
