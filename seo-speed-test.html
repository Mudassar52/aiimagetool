FAQ

Frequently Asked Questions

What is PageSpeed Insights score?
Google PageSpeed Insights scores your website 0–100 for both Mobile and Desktop. A score of 90+ is considered Good, 50–89 is Needs Improvement, and below 50 is Poor. The score is based on real Chrome user data and lab metrics like LCP, FID, CLS.
What does the SEO audit check?
The SEO audit checks: Title Tag (length, existence), Meta Description (length, existence), H1–H6 Heading structure, Keyword Density of top words, Image ALT attributes, Canonical Tag presence, and more — all in one report.
Is this tool free?
Yes, completely free. The PageSpeed score uses Google's public PageSpeed Insights API, and the SEO scan runs directly in your browser.
Why does the SEO scan use AI?
Browsers cannot directly fetch other websites due to CORS restrictions. We use Claude AI with web search capability to fetch and analyze the live page — extracting meta tags, headings, image ALTs, keywords, and more in real time.
